# 産卵床調査アプリ

サケ科魚類の産卵床（レッド）調査を、現場のスマホだけで記録するアプリ。
HTMLファイル1つで動き、電波が無くても使えます。記録はスマホの中だけに保存され、
外へ自動送信することはありません。

## できること

- 河川の**区間を歩きながら**GPSで軌跡を取り、区間長を自動で出す（手入力で直すことも可）
- **産卵床**を1件ずつ記録：位置・番号・魚種・新旧・状態・長さ幅（面積は自動計算）・
  水深・流速・底質・埋没度・被覆・河川水温・河床内水温・親魚（雌雄・行動）・ホッチャレ数
- 「保存して次を記録」で、区間・魚種・底質・水温を引き継いで**連続入力**
- **声で入力**（「水深25 流速0.4 中礫 サケ 新しい」と話すと該当欄へ入る）
- **写真**を産卵床やメモに添付（長辺1600pxに縮小して保存）
- **調査メモ**（調査条件・河川状況・特記事項・次回課題）
- **地図**で確認（航空写真／地形図。魚種で色、新旧で濃さ、区間は線）
- **集計**（全体と区間別。密度＝床数/100m）
- **書き出し**：CSV（Excel用・BOM付き）／GeoJSON（QGIS用・EPSG:4326）／写真ZIP／バックアップ

## ファイル

| ファイル | 役割 |

## 動かす

`file://` で直接開くと動きません。フォルダ内で簡易サーバを立ててください。

```
"C:\Users\lysoc\AppData\Local\Programs\Python\Python312-arm64\python.exe" -m http.server 8765
```

→ `http://127.0.0.1:8765/redd_survey.html`

## 動作確認

```
node tests/run_all.js
```

保存・区間の距離計算・面積計算・写真・集計・CSV/GeoJSON・バックアップ復元・
声の聞き取りまで、まとめて確認します。

## 保存先について

- 記録本体：ブラウザの localStorage（キーは `rd_` で始まる）
- 写真：ブラウザの IndexedDB（`redd_survey_media`）
- 同じ場所に置いた他のアプリ（鳥類調査アプリなど）とはキャッシュも保存先も分けてあります
